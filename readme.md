# android-traffic-exchange
Traffic Exchange App For Android

## Features

- [x] unique visitor
- [x] override useragent
- [x] WebRTC
- [ ] override referer
- [ ] override custom header
- [ ] override proxy

## Requirements
- Android 11
- Internet Connection
- Google Account

## Develop Requirements
- JDK-17
